## 🔄 Authentication Flow

```
Registration:
┌─────────────────────────────────────────────────────────────┐
│ 1. User fills register form (name, email, password)        │
│    ↓
│ 2. Client validates (regex, length, required)             │
│    ↓
│ 3. Submit → POST /auth/register                           │
│    ↓
│ 4. Backend validates & creates user                        │
│    ↓
│ 5. Returns: access_token, refresh_token, user data        │
│    ↓
│ 6. Frontend stores tokens in localStorage                  │
│    ↓
│ 7. Updates auth context (user = true)                      │
│    ↓
│ 8. Redirects to /dashboard                                 │
└─────────────────────────────────────────────────────────────┘

Login:
┌─────────────────────────────────────────────────────────────┐
│ 1. User enters email & password                            │
│    ↓
│ 2. Client validates                                        │
│    ↓
│ 3. Submit → POST /auth/login                              │
│    ↓
│ 4. Backend validates credentials                           │
│    ↓
│ 5. Returns: access_token, refresh_token, user data        │
│    ↓
│ 6. Same flow as registration (steps 6-8)                  │
└─────────────────────────────────────────────────────────────┘

Logout:
┌─────────────────────────────────────────────────────────────┐
│ 1. User clicks logout button                               │
│    ↓
│ 2. Clear tokens from localStorage                          │
│    ↓
│ 3. Reset auth context (user = null)                        │
│    ↓
│ 4. Redirect to /login                                      │
└─────────────────────────────────────────────────────────────┘

Protected Route:
┌─────────────────────────────────────────────────────────────┐
│ 1. User tries to access /dashboard                         │
│    ↓
│ 2. Check if user in context                                │
│    ↓
│ If YES: Render dashboard                                   │
│ If NO: Show loading spinner                                │
│ If TIMEOUT: Redirect to /login                             │
└─────────────────────────────────────────────────────────────┘
```
